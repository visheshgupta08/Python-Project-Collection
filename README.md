# Python Projects Collection 🐍🚀

Welcome to my Python repository! This repository is dedicated to building, testing, and storing various Python programs and projects, ranging from basic concepts to advanced logic.

## 📁 Featured Projects
Here are the core logical projects included in this repository:

1. 🏆 **Kaun Banega Crorepati (KBC) Clone** - Core logic, conditional execution, and list management.
2. 📚 **Library Management System** - Complete OOPs-driven application featuring automatic index tracking, instance encapsulation, and list-lookup validation to prevent duplicate records.
3. 📄 **PDF Merger Utility** - Working with external libraries (`PyPDF2`) for real-world file manipulation.
4. 🧹 **Clear the Clutter** - Automation script using the `os` module to organize local directory systems.
5. 🎮 **Snake, Water, Gun Game** - Interactive game mechanics using random module logic.
6. 💸 **Personal Budget Tracker** - Data-handling program utilizing dictionaries and json persistence for safe financial recording.
7. 📰 **Live News Fetcher Utility** - API automation tool using requests module to fetch, parse, and filter real-world global headlines in JSON format.
8. 🥤 **Drink Water Reminder** - OS-level background automation script utilizing pyttsx3 and Windows scheduler logic for timely voice-based user reminders.
9. 🔐 **Secret Code Language Converter** - Text manipulation utility featuring string slicing, indexing, and random letter padding to encrypt and decrypt hidden messages.
10. 🔊 **Bulk Shoutout Automated Engine** - Audio synthesis automation program using the pyttsx3 text-to-speech engine to process and voice out batch lists dynamically.

## 🛠️ Requirements & Installation
To run any script locally, make sure you have Python installed:
1. Clone this repository or download the files.
2. Open your terminal in this folder.
3. Run the script using:
```bash
python filename.py
```
